
## Laser cut snap-fit box

I created a box on illustrator. I made the diagram from scratch without using any reference. 

## Sizes for the box
* Height - 10cm
* width - 10cm
* Depth - 5cm

* MDF thickness - 3mm
* knotch length - 10mm
* kerf value - 0.2mm
